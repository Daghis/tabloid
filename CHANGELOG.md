# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- GitHub Actions CI/CD workflows (test, lint, deploy)
- ESLint configuration for code quality
- Test infrastructure with test runner
- CONTRIBUTING.md with contribution guidelines
- Issue and PR templates
- CHANGELOG.md for tracking changes
- Comprehensive .gitignore file

### Changed
- Enhanced package.json with repository metadata

## [1.0.0] - 2020

### Added
- Initial release of Tabloid programming language
- Web-based interpreter with Torus UI
- Node.js REPL
- Core language features:
  - Functions with recursion
  - Conditional expressions
  - Arithmetic and boolean operators
  - Variables
  - Print and input operations
  - Block comments
- MIT License
- README with language documentation
